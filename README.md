## 📢소개📢
expo, react-native, firebase을 활용하여 만든 채팅앱입니다.<br/>
firebase에서 로그인 회원가입을 구현했고, firebase database에 채팅한 목록이 저장됩니다.<br/>
랜덤채팅은 아니고, 그냥 giftedchat을 이용해 만든 나랑 톡하기 같은 채팅앱입니다.
